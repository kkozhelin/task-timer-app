# Task Manager with Timer

## Project Overview

This project is a simple **Task Manager** built with React and TypeScript. It includes functionality to create, update, toggle, and delete tasks. Additionally, there is a built-in **Timer** feature to help users focus on tasks. The application supports both light and dark themes, providing a better user experience.

## Features

- **Task Management**:
  - Add tasks with a unique ID.
  - Toggle task completion status.
  - Delete tasks.
- **Timer**:
  - Start, pause, and reset a timer.
  - Track time for focused work sessions.
- **Theming**:
  - Switch between light and dark modes.
  - Tailwind CSS for styling.
- **State Management**:
  - Redux Toolkit for centralized state management.
- **Responsive Design**:
  - Fully responsive layout for desktop and mobile devices.

## Technologies Used

- **React**
- **Redux Toolkit**
- **TypeScript**
- **Tailwind CSS**

## Installation

1. Install dependencies:
   ```
   npm install
   ```
2. Start the development server:
   ```
   npm run dev
   ```
3. Open the application in your browser at:
   ```
   http://localhost:3000
   ```
